# SIPENAB Project Rules

## Project Overview
SIPENAB (Sistem Penilaian Beasiswa PNJ) is a web-based decision support system for scholarship selection using the SAW method.

## Mandatory Conventions
- **Framework**: CodeIgniter (v3 or v4 - verify version in project).
- **Architecture**: Model-View-Controller.
- **Naming**: Use camelCase for variables and PascalCase for Classes.
- **Calculation**: Method SAW (Normalization -> Preference -> Ranking).

## Verification Protocol
- All new features must be verified with functional tests.
- Calculation results must be cross-checked against the SAW mathematical formula.
